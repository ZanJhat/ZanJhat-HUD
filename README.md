# ZanJhat HUD

A customizable HUD and tooltip overhaul mod for a better UI experience.

---

## Features
- Display coordinates
- Display world time information (day, hour, rain time, fog, etc.)
- Clothing information
- Show item info when hovering slots or dragging items on mobile
- Quick recipe and usage viewing

---

## Requirements
- ZanJhat Core >= 2.0.0

---

## Compatibility
- API Version: >=1.9.0.2
- SC Version: 2.4.0.0

---

## Installation
1. Install ZanJhat Core first
2. Install ZanJhat HUD
3. Ensure correct load order
